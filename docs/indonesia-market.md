### 인도네시아 하이퍼로컬 페인포인트

1. **기존 커뮤니티의 디지털화 부재**
   - 동네 WhatsApp 그룹은 있지만 체계화된 플랫폼 없음
   - 동네 소식이 카카오(WhatsApp)에 묻혀 검색 불가

2. **중고거래 신뢰 문제**
   - OLX: 사기 거래 빈번, AI 검수 없음
   - Tokopedia: 새 제품 중심, 지역 중고 약함

3. **구인구직 하이퍼로컬 공백**
   - JobStreet/LinkedIn: 대기업 중심
   - 동네 편의점·식당·아르바이트 채용 채널 없음

4. **지역 소상공인 디지털 전환 미흡**
   - 인스타그램 페이지만 운영, 통합 관리 도구 없음

## 경쟁사 심층 분석

### Gojek (가장 강력한 경쟁자)
- **강점**: 슈퍼앱 1위, 배달·교통 생태계, GoTo 그룹
- **약점**: 하이퍼로컬 커뮤니티 없음, 중고거래 없음
- **Mozzy 기회**: 커뮤니티·중고·부동산 영역 공략

### OLX Indonesia
- **강점**: 중고거래 인지도 1위
- **약점**: AI 검수 없음, 오래된 UX, 커뮤니티 없음
- **Mozzy 기회**: AI 검수 + Trust Score 차별화

### Kaskus
- **강점**: 커뮤니티 문화 강함, 레거시 브랜드
- **약점**: 노후한 모바일 앱, Z세대 이탈
- **Mozzy 기회**: 현대적 UX + 하이퍼로컬 포커스

## GTM 실행 계획

### Month 1-3: 반둥 집중 공략
```
채널 믹스:
- 반둥 대학교 캠퍼스 오프라인 부스 (ITB, UNPAD, Telkom University)
- Instagram/TikTok 반둥 로컬 인플루언서 KOL 10명
- "Jual Beli Aman di Bandung" 캠페인
- 첫 AI 검수 무료 프로모션 (5,000건)

목표: 반둥 DAU 5,000 / Marketplace 거래 500건/월
```

### Month 4-6: 자카르타·수라바야 확장
```
채널 믹스:
- Business+ 무료 3개월 (첫 500개 가게)
- 부동산 중개업자 파트너십 (자카르타)
- "Cari Kerja di Sekitar Kamu" (Jobs 캠페인)
- Google Ads (인도네시아 키워드: jual beli bekas, lowongan kerja sekitar)

목표: 3개 도시 합산 DAU 25,000
```

### 인도네시아 바이럴 루프 설계
```
1. 물건 팔기 → AI 검수 → 빠른 거래 → 친구 추천
2. 동네 뉴스 → 댓글 → 이웃 알림 → 재방문
3. 구인 → 채팅 → 거래 → 가게 팔로우
4. POM(뽐) → 공유 → 외부 유입 → 가입
```
